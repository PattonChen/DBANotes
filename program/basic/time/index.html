常用方法：

time.time()

显示当前时间戳。

>>> time.time()
>>> 1595662634.6728134

显示为整数的秒。

>>> int(time.time())
>>> 1595662640

time.localtime([Seconds])

将时间戳转换成“当前时区”的struct_time（结构化数据）。显示为时间数组模式的时间。

参数：Secs –秒数[默认为time.time()]，用来转换为time.struct_time类型的对象

  • 给定参数
>>> time.localtime(time.time())
>>> time.struct_time(tm_year=2020, tm_mon=7, tm_mday=25, tm_hour=15, tm_min=40, tm_sec=50, tm_wday=5, tm_yday=207, tm_isdst=0)
  • 不给参数
>>> time.localtime()
>>> time.struct_time(tm_year=2020, tm_mon=7, tm_mday=25, tm_hour=15, tm_min=40, tm_sec=38, tm_wday=5, tm_yday=207, tm_isdst=0)

调用其中的数据,如年，月。

>>> time.localtime().tm_year
>>> 2020
>>> time.localtime().tm_mon
>>> 7

time.gmtime([Secs])

与time.localtime()类似，但此处为将时间戳转换成UTC时区（0时区）的struct_time对象（struct_time是time模块中定义的对象）。

参数Secs：可以是从1970-1-1 00:00:00以来的秒数。默认为当前的时间戳time.time()

也可以类似上面的来调用struct_time的属性。

>>> time.gmtime()
time.struct_time(tm_year=2020, tm_mon=7, tm_mday=25, tm_hour=9, tm_min=2, tm_sec=16, tm_wday=5, tm_yday=207, tm_isdst=0)

>>> time.gmtime(1595662634.6728134)
time.struct_time(tm_year=2020, tm_mon=7, tm_mday=25, tm_hour=7, tm_min=37, tm_sec=14, tm_wday=5, tm_yday=207, tm_isdst=0)

>>> time.gmtime(time.time());
time.struct_time(tm_year=2020, tm_mon=7, tm_mday=25, tm_hour=9, tm_min=3, tm_sec=3, tm_wday=5, tm_yday=207, tm_isdst=0)

time.mktime(t)将stuct_time转化成时间戳

time.mktime是gmtime(),localtime()逆向操作。stuct_time转化成时间戳

参数：t -结构化时间，或者完整的9位元组元素。

>>> time.localtime()
time.struct_time(tm_year=2020, tm_mon=7, tm_mday=25, tm_hour=17, tm_min=8, tm_sec=43, tm_wday=5, tm_yday=207, tm_isdst=0)
>>> time.mktime(time.localtime())
1595668133.0

time.sleep(Secs) 线程推迟指定的时间后再运行。睡眠时间

参数： Secs -秒数，type : float or int
