乙醛脱氢酶

  • ALDH 属于醛脱氢酶的一种。负责催化乙醛氧化成乙酸的反应。化学式：CH3CHO + NAD+ + CoA ↔ 乙酰CoA + NADH + H+
  • 乙醛脱氢酶由三个基因所编码：
  • ALDH1A1（位于胞质溶胶），
  • ALDH2（位于线粒体）
  • ALDH5（ALDH1B1，最近发现）

CH3CHO + NAD+ + CoA ↔ 乙酰CoA + NADH + H+

分别分布於胞质溶胶（ALDH1）与线粒体（ALDH2）
