mysql> insert into stu (name,gender) values('b',2);
ERROR 1364 (HY000): Field 'id' doesn't have a default value
  • 指定非空时，会报错某列不允许为空。
  • 如果定义字段为非空遇到已存在非空值时，就会报错。

Modify table

  • 修改列的属性，非空，默认值，数据类型，数据的长度and so on.
  • 语法：ALTER TABLE tbl_name modify tbl_column_name DATA_TYPE(DATA_LENGTH) ...
mysql> alter table stu modify id int(11) not null default 11;
