    专注自律

    1. 干净学习环境，去掉外部干扰。

    1. 所有容易造成分神的，与学习无关的东西移开。
    2. 社交媒体，手机或者电脑的即时通讯工具
    3. 不高估专注程度与自制力。容易受到干扰。
    4. 专注程工特点，除了自动聚集到一点外，自下而上的一个驱动，容易受到“叮”的声音干扰。
    5. 分心的事情暂不处理，记录在todolist上面。

    2.有意识训练自己的集中精神的时长（专注）时间。去除内部干扰.

    1. 循序渐进训练！从5到45分钟
    2. 休息5分钟，给大脑一个奖励奖赏。吃点什么，看新闻等。
    3. 注意力就是智力的一部分，且可以通过训练得以巩固。
    4. 去除内部干扰.
    5. 心不念未来的事情，只看当下，现在。
    6. 列计划清单，当完成任务后划掉。

    3. 学习仪式感建立，渐渐进入学习意境。

    1. todolist，完成一个划掉一个。

    2. 进行这个仪式就自动进入学习的状态。

    监控大脑，触发学习开关

    多感官刺激，拉加注意力。

    4. 善于用好现有的工具，不要忌讳使用工具。

    1. 用好现在的工具，提交效率
    2. 忌为找工具面浪费时间。
    3. 计时学习与休息。

    5. 训练注意自己要走神能力

    1. 注意到自己是否走神。
    2. 衡量走神的原因是什么？

    6. 坚持完成一个事情后，形成心流。

    完成30-45分钟事情，期间会进入心流的状态。越能体会到这点，越能走近学学霸。
